# cncec16
本系统基于SSM+Jquery制作，前端采用Bootstrap框架，负责公司信息，人员管理等

# 功能预览
采用Spring,Spring Mvc, Mybatis, Bootstrap，jquery框架制作，数据库采用Mysql, 前台展示公司基本消息界面，内容界面。后台由管理员，超级管理员构成，
具有有管理用户增删改查、消息上传，文件上传下载等多个功能。
